### advanced feature

#### `shift` 函数

- 可以变换rgb通道为别的排列
- 这使得不用特别大尺寸的马赛克也能抹掉头像特征成为可能

#### 混合多种插值方式的降采样

- 创建了一个预设，用少许最邻近采样来抹掉一些特征
- 这使得不用特别大尺寸的马赛克也能抹掉头像特征成为可能
